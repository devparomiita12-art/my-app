# User Stories

1. As a user, I want to sign up so I can create an account.
2. As a user, I want to log in so I can access my account.
3. As a user, I want to view the home page.
4. As a user, I want to view item details.
5. As a user, I want to save favorites.
6. As a user, I want to manage my profile.
7. As a user, I want to access settings.
8. As a user, I want to receive notifications.
9. As a user, I want to log out.
